# SAP Computer Project Conventions

## Component Orientation

| Component | Facing |
|------------|---------|
| Program Counter | East |
| Memory | East |
| Instruction Register | East |
| Register A | West |
| Register B | West |
| Adder/Subtractor | West |
| Controller | North |
| Clock | South |

## Color Scheme

| Category | Color | Hex |
|----------|--------|--------|
| Registers (PC, IR, A, B) | Blue | #5B9BD5 |
| Memory | Green | #70AD47 |
| Adder/Subtractor | Orange | #ED7D31 |
| Controller | Dark Red | #C00000 |
| Clock | Purple | #7030A0 |
| Bus/Wiring | Dark Gray | #7F7F7F |
